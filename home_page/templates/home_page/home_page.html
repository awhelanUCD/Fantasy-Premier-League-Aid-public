Player Data

Make informed transfer decisions and team selections by using this feature. A table with both basic and advanced data such as goals, assists and expected goals is provided. This table has search and sort functionality and allows users to add and remove columns so only the data that they want to see is shown.

View Player Data

Live Gameweek

Players of head-to-head fantasy leagues no longer have to wait until the end of the gameweek to see updated league standings. Real-time league standings, player data and fixture scores are provided.

View Live Gameweek

League Table

View the league table of your fantasy league with additional information. Namely, a form column and a global percentile column. For players of head-to-head leagues, the form column gives results from each players last five fixtures. For classic leagues, the average points from the last five gameweeks is provided.
